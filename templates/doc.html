What are Favicons?

Favicons are used in appbars, browser history, bookmark dropdown, browser tab, search bar etc. It is a little icon that appears next to the webpage title in a browser tab to represent a website or web page. It serves as a visual identifier for users to spot your website around the web.

Why You Need a Favicon?

From research carried out by our team, we found that often-times users assume websites without favicons are unsafe and they are unprotected while on these sites. Having favicons on your websites not only allows your users easily identify your website or brand, it also builds a sense of trust in anyone that views your website. And so, for a better brand identification and trust building, you need a favicon.

Sizes in Which Favicon Will Be Displayed

  • Browser favicons - 16x16
  • Taskbar shortcut icons - 32x32
  • Desktop shortcut icons - 96x96
  • Chrome for Android - 160x160
  • Chrome web store icon - 128x128
  • Apple Touch icon-180x180


Favicon File Format

  • SVG: It was not widely supported on browsers, but things are moving in a positive direction. Browsers can also display GIFs as favicons, but they are hard to see. SVG files are known to deliver high-quality images without compromising the look, speed, or performance of the site.
  • PNG: PNG files are very popular for creating favicons, as it's a widely used file format used by creators. They are lightweight and load quickly. When using designs with transparent backgrounds, this is the format to use.
  • JPG: JPG files are not usually used for favicons, as they don't have the same high-level quality as a PNG.
  • Windows ICO: This was the original file type and is still used by designers. This file type can have multiple bit depths and resolutions.
